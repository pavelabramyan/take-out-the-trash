# ВЫНЕСИ МУСОР! / TAKE OUT THE TRASH!

Короткая мем-игра для Steam: мама крикнула — ты несёшь физический пакет через подъезд панельки. Пакет рвётся, соседи смотрят, во дворе собаки и лёд.

**Цена:** $2.99 · **Объём:** 12 уровней, ~60–90 мин · **Движок:** Godot 4.4 · **Языки:** RU + EN

## Запуск

```bash
./Godot.app/Contents/MacOS/Godot --path .
```

Или открой папку в Godot 4.4 → Import → Play (F5).

## Управление

- WASD — ходить · мышь — смотреть · Shift — бег · Ctrl — присед · Space — прыжок
- E — выбросить в помойку / лифт / поднять мусор
- F — фонарик телефона · R — рестарт · Esc — пауза

## Механики

- **Пакет:** RigidBody в руках, урон от ударов и падений → разрыв → куски мусора → собери и донеси
- **Звёзды:** быстро / без разрывов / стелс от бабушек и собак
- **Эскалация:** пакет → вёдра → ковёр → холодильник; лифт-лотерея, ночь, лёд, ветер, собаки

## Структура

- `scripts/player.gd` — FPS
- `scripts/trash_bag.gd` — физика груза и разрыв
- `scripts/building_builder.gd` — подъезд + двор
- `scripts/level_data.gd` — 12 уровней
- `scripts/game.gd` — цикл уровня
- `scripts/main_menu.gd` — меню / настройки / сейвы
- `tools/` — генераторы SFX/текстур, quality gate, store assets

## Проверки

```bash
./tools/quality_gate.sh quick
./tools/quality_gate.sh full   # + export, если есть шаблоны
```
