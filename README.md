# ВЫНЕСИ МУСОР! / TAKE OUT THE TRASH!

Короткая **chore physics comedy** для Steam: мама крикнула — ты несёшь физический пакет через подъезд. Пакет рвётся от ударов, соседи смотрят.

**Цена:** $2.99 · **Объём:** ~1–2 часа (12 уровней) · **Движок:** Godot 4.4 · **Языки:** RU + EN

## Запуск

```bash
./Godot.app/Contents/MacOS/Godot --path .
```

## Управление

- WASD — ходить · мышь — смотреть · Shift — бег · Ctrl — присед · Space — прыжок
- **E** — поднять пакет / мусор / лифт / помойка
- **ЛКМ** — бросить · **ПКМ** — аккуратно положить · **Alt** — режим «аккуратно» (меньше урон)
- F — фонарик · R — рестарт · Esc — пауза · P — photo marker

## Механики

- Пакет с **коллизией в руках** — удары о перила снимают HP
- Разрыв → охапка кусков → к помойке
- Звёзды: время / целый пакет / стелс
- Эскалация: thin → лифт → ночь → подвал → лёд → ветер → собаки → ковёр → холодильник
